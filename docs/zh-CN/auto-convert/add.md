# 创建自动转换



```
/rate auto add A to B
/ra auto add A to B
/创建转换 A to B
```

A B 为 [货币代码](../glossary/currency-code.md) 或 [别称](../glossary/alias.md)

## 用法

```
自动识别并转换 美元 为 人民币
/rate auto add 美元 to 元

自动识别并转换 日元 为 土耳其里拉
/rate auto add JPY to TRY

自动识别转换 欧元 为 基准货币
/ra auto add EUR
```
