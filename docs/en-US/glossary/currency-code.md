# 货币代码

Currency Converter 使用 [_ISO 4217 Three Letter Currency Code_](https://en.wikipedia.org/wiki/ISO\_4217) 作为货币代码，你可以在 [此处](https://www.exchangerate-api.com/docs/supported-currencies) 查看支持的所有货币。

常用的货币及代码如下：
